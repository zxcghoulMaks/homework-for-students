# React Native

Цей репозиторій містить домашні завдання з JavaScript, TypeScript, React та React Native.

## Як форкнути репозиторій

1. Натисніть кнопку **Fork** у правому верхньому куті цього репозиторію.  
2. Виберіть свій акаунт GitHub. Тепер у вас з’явиться копія репозиторію у вашому акаунті.

## Як запулити репозиторій до себе

Відкрийте термінал і виконайте команду:  
```bash
git clone <URL_вашого_форкнутого_репозиторію>
```
## Структура репозиторію

### 1. js-hw
- Містить 10 задач по JavaScript.
- Всі завдання написані прямо у файлах.  
- У кожному файлі треба реалізувати рішення задачі.

### 2. ts-hw
- Містить 8 задач по TypeScript.
- Завдання також прописані у файлах.  
- У кожному файлі потрібно написати реалізацію задачі.

### 3. react-hw
- Проект на React.  
- У файлі App.jsx потрібно виконати завдання.
- Щоб запустити проект:
  1. Перейдіть у папку react-hw.
  2. Виконайте команду для встановлення залежностей: npm install
  3. Запустіть проект: npm run dev
  4. У консолі з’явиться посилання, наприклад http://localhost:5173/. Натисніть на нього — відкриється сайт, над яким ви працюєте.

### 4. react-native-hw
- Містить файл App.jsx.  
- У нього потрібно вставити рішення завдання.  
- React Native встановлювати на комп’ютер не потрібно.  
- Достатньо скористатися [Expo Snack](https://snack.expo.dev/) для запуску та перевірки коду.

## Як завантажити свої рішення назад

1. Додайте змінені файли до індексу Git:
```bash  
git add .
```
3. Зробіть коміт із описом:
```bash
git commit -m "Рішення домашніх завдань"
```
4. Відправте зміни на свій форк:
```bash
git push origin main
```
